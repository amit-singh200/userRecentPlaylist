#User's Recent Songs Playlist README file.

Follow the below steps on how to execute the code.

1. Install the Pycharm(Community Edition -IDE) or 
   any other Python compatible IDE.
2. Download the Python3.8.5.

3. Install the Python3.8.5 and add the Python3.8.5 path to the PATH var while instaling.
4. Configure the IDE i.e. Pycharm or other IDE to run the Python code. 
    i.e. add interpreter as Python3.8.5 under config settings.

5. Create the Python project(e.g. UserSongs) under the IDE.(Virtual environment- Optional).
6. Add the three (3) .py python files (i.e. main.py, userPlaylist.py and testUserPlaylist.py) to the project.

7. Execute the main.py file by right click on the main.py and select option Run 'main'.
   Also we can modify the input parameters in main.py under UserPlaylist class objects.

8. To run/execute the unittest file (i.e. testUserPlaylist.py) with unit tests Right click the file testUserPlaylist.py and 
   select option Run 'Python tests in testUserPlaylist'
